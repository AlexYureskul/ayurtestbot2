Описание
"Моя Телеграм Игра" — это мини-игра-кликер с 3D-графикой, в которой игроки могут выбирать персонажей, зарабатывать очки, приглашать друзей и взаимодействовать с ними. Приложение поддерживает несколько языков и адаптировано для мобильных устройств.

Установка и запуск
1. Клонирование репозитория
bash
Копировать код
git clone https://github.com/yourusername/my_telegram_webapp.git
cd my_telegram_webapp
2. Настройка Firebase
Создайте проект в Firebase и включите анонимную аутентификацию и Firestore Database.
Создайте файл config.js на основе config.example.js и вставьте в него вашу конфигурацию Firebase.
Добавьте config.js в .gitignore.
3. Настройка бота
Требования
Python 3.7 или выше
Установленные библиотеки:
python-telegram-bot==20.3
firebase-admin
Шаги настройки
Установка необходимых библиотек

bash
Копировать код
pip install python-telegram-bot==20.3 firebase-admin
Получение учетных данных сервисного аккаунта

Перейдите в консоль Firebase и скачайте файл serviceAccountKey.json.
Поместите файл в директорию с bot.py.
Добавьте serviceAccountKey.json в .gitignore.
Обновление bot.py

Замените 'path/to/serviceAccountKey.json' на путь к вашему файлу serviceAccountKey.json.
Замените 'YOUR_TELEGRAM_BOT_TOKEN' на токен вашего бота.
Замените yourusername на ваш GitHub username в ссылках.
Запуск бота

bash
Копировать код
python bot.py
4. Размещение на GitHub Pages
Загрузите файлы в репозиторий GitHub.
Включите GitHub Pages в настройках репозитория, выбрав ветку main и папку /root.
Ваше приложение будет доступно по адресу https://yourusername.github.io/my_telegram_webapp/.
5. Тестирование приложения
Откройте бота в Telegram и отправьте /start.
Нажмите на кнопку "Открыть игру" и начните играть.
Функциональность
Кликерная механика: Нажимайте на кнопку, чтобы зарабатывать очки. Кнопка будет динамически перемещаться после нажатия.
Выбор персонажа: Выбирайте из 5 различных персонажей в разделе настроек.
Хаотично движущиеся модели: На фоне отображаются модели, движущиеся по экрану.
Реферальная система: Приглашайте друзей и получайте бонусные очки.
Список друзей: Просматривайте список друзей, которых вы пригласили.
Локализация: Приложение поддерживает русский, английский и украинский языки.
Настройки: Настраивайте язык и персонажа в разделе настроек.
Использование
Начало игры
Запустите бота и нажмите на кнопку "Открыть игру".
Игра откроется внутри Telegram.
Заработок очков
Нажимайте на кнопку с надписью "Нажми меня!", чтобы заработать очки.
Кнопка будет динамически перемещаться после нажатия.
Выбор персонажа
Нажмите на иконку настроек (шестерёнка) в верхнем левом углу.
Выберите желаемого персонажа и нажмите "Закрыть".
Приглашение друзей
Используйте команду /referral в боте, чтобы получить вашу реферальную ссылку.
Отправьте ссылку друзьям. Когда они начнут играть, вы получите бонусные очки, а друг появится в вашем списке друзей.
Просмотр друзей
Нажмите на иконку друзей (два человечка) в верхнем левом углу.
Откроется список друзей, которых вы пригласили.
Также можно использовать команду /friends в боте, чтобы увидеть список друзей.
Переключение языка
Нажмите на кнопки "English", "Русский" или "Українська" в верхнем правом углу, чтобы изменить язык интерфейса.
Команды бота
/start — Начать работу с ботом и получить ссылку на игру.
/help — Получить справочную информацию.
/referral — Получить вашу реферальную ссылку для приглашения друзей.
/friends — Просмотреть список приглашённых вами друзей.
